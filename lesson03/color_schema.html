Color Schema

Here's a link to the palette

The application of the color scheme is as follows:

  1. #222222: Used for the footer.
  2. #42515A: Applied to headings.
  3. #F9F7F3: Utilized for navigation hover effects.
  4. White: Employed for font color in the header, footer, and navigation items.
  5. Black: Used as the main font color.
  6. Black, green, and yellow: Designated as the logo colors.
